# tick

CLI tool to poll a stock symbol, say something when certain states change.

# Usage

```sh
npm install
./index.js --symbol GOOG --lower 500 --upper 600
```

# Data Source

Uses the undocumented and deprecated Google Finance API.  This script will stop
working at any time.
